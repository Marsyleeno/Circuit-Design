# Circuit-Design
BJT amplifier and Logic Module

## Test Hardware for Digital Logic Gates

<p>
  This project is a custom-designed digital logic training module created to facilitate hands-on laboratory experiments with combinational and sequential circuits. The board features an input section utilizing multiple toggle switches to manually simulate high and low logic states for testing various gate configurations. For output visualization, it integrates an array of eight discrete LEDs alongside a 7-segment display to clearly indicate the resulting logic states of the circuits under test. Additionally, an onboard NE555 timer circuit is included to serve as a reliable clock pulse generator, which is essential for driving sequential logic components like flip-flops and registers.
</p>

## Common Emitter Amplifier for Small Signal Analysis

<p>
  A practical breadboard implementation of a two-stage discrete transistor audio amplifier designed specifically for small signal analysis. The first stage utilizes a 2N3904 NPN transistor configured as a common-emitter amplifier, which provides the primary voltage gain necessary to boost the incoming low-level audio signal. Following this, the second stage employs a 2N2222 transistor in a common-collector, or emitter-follower, configuration to provide crucial current gain and impedance matching. An input potentiometer allows for manual attenuation of the source signal, while the final output directly drives a small 8-ohm speaker.
</p>
