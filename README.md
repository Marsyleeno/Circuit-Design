# Circuit-Design
BJT amplifier and Logic Module

-- L O G I C    M O D U L E--
<p>
  This project is a custom-designed digital logic training module created to facilitate hands-on laboratory experiments with combinational and sequential circuits. The board features an input section utilizing multiple toggle switches to manually simulate high and low logic states for testing various gate configurations. For output visualization, it integrates an array of eight discrete LEDs alongside a 7-segment display to clearly indicate the resulting logic states of the circuits under test. Additionally, an onboard NE555 timer circuit is included to serve as a reliable clock pulse generator, which is essential for driving sequential logic components like flip-flops and registers.
</p>
